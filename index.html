Summary: Latitude vs. Weather Data

The purpose of this project was to analyze how weather Changes as it gets closer to the equator. To accomplish this analysis, the first step was to pulled data from the OpenWeatherMap API to assemble a dataset on over 500 randomly selected cities.

The second step was applying Matplotlib to plot various aspects of the weather data vs. latitude. Factors considered including: temperature, cloudiness, wind speed, and humidity. This site provides the source data and the visualizations results of this analysis, as well as explanations and descriptions of any trends and correlations witnessed.
